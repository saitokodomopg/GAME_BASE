# ゲーム開発テンプレート

Claude Code を使ってブラウザゲームを作るためのひな型です。

---

## 開発を始める前に

このテンプレートでは **Claude Code（AIアシスタント）** と一緒にゲームを作ります。  
Claude はコードを書くだけでなく、設計の相談や仕様の整理も手伝ってくれます。

---

## 手順

### 1. このリポジトリをコピーする

GitHub の「Use this template」ボタン、またはダウンロードして自分のフォルダに置いてください。

### 2. Claude Code でフォルダを開く

Claude Code（CLI またはエディタ拡張）で、コピーしたフォルダを開きます。

### 3. ゲームの内容を決めて、Claude に伝える

まずゲームのアイデアを Claude に話してください。  
Claude が一緒に整理してくれます。

> 例：「横スクロールアクションゲームを作りたい。主人公がジャンプして障害物を避けるゲーム」

Claude は以下のファイルを順番に作成します（各ファイルの作成前に内容を確認してください）：

| ファイル | 内容 |
|----------|------|
| `設計/concept.md` | ゲームのコンセプト・仕様概要 |
| `設計/architecture.md` | 使う技術・設計方針 |

### 4. 作業単位を決めて実装を進める

「今日はここまで作る」という単位を Claude に伝えると、  
Claude が `作業メモ/` フォルダに作業計画ファイルを作成します。

> 例：「まずキャラクターが画面に表示されて左右に動けるところまで作りたい」

作業計画が作成されたら内容を確認し、問題なければ実装を進めます。

### 5. 繰り返す

手順 4 を繰り返して機能を追加していきます。

## フォルダ構成

```
リポジトリフォルダ/
├── README.md              # このファイル（人間向けの説明）
├── CLAUDE.md              # Claude 向けのルール（基本的に変更しない）
├── import/                # 参考資料を置く場所（読み取り専用）
├── 設計/                  # ゲームの仕様・設計を記録するファイル
│   ├── concept.md         # ゲームコンセプト
│   ├── architecture.md    # 技術スタック・設計の決定事項
│   └── known-issues.md    # あとでなおしたいことのメモ
├── 作業メモ/              # 作業単位ごとの計画ファイル（Claude が作成）
└── src/                   # ゲームのソースコード
```

## よくある質問

**Q. `CLAUDE.md` は何ですか？**  
A. Claude に対するルールが書かれたファイルです。基本的に変更不要です。

**Q. `設計/` のファイルは自分で編集してもいいですか？**  
A. Claude に頼んで一緒に編集するのがおすすめです。直接編集する場合も、大きな変更は Claude に相談してから行ってください。

**Q. `作業メモ/` フォルダに何が増えていきますか？**  
A. 作業単位ごとに `20260529-1000-キャラクター移動/` のようなフォルダが増えます。完了した作業も残るので、後から振り返ることができます。

**Q. 途中でゲームの内容を変えたくなったら？**  
A. Claude に相談してください。小さな変更は `設計/concept.md` を更新し、大きな方向転換の場合は新しいファイルで議論してから反映します。
